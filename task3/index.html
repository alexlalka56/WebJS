HTML tags

If not taking into account the base HTML skeleton, the HTML document allows us to use many special elements in it. There are hundreds of elements you can use. However, in your work, you will use fewer elements. The name of these elements is Tags.

A basic HTML document is just a document that you can see over the internet, like Microsoft Office. It has titles, paragraphs, images, bold text, italic texts, or text without any formatting, links, lists, etc.

  • Titles are: h1-h6
  • Paragraph is: p
  • Image: img
  • Bold text is: strong or b
  • Italic is: i
  • Text without any formatting is: span
  • Links: a
  • Buttons: button
  • Form and form elements: form, input, textarea
  • Lists are: ul or ol

These elements can be placed somewhere on the page. And to build the places on the page we can build layout using blocks.
Block tag is: div
In this example we use two blocks. The first block div id="app" is main application block that is root of our application.
This block contains main title h1 and block div class="article" that contains
another type of title h3 and other tags p, strong, br
